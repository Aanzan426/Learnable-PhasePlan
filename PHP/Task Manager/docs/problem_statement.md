## 5️⃣ Difficulty 5 — Task List Manager (DONE)



### Problem Statement



Build a task management application that allows users to add tasks, mark them as complete, and delete them, with persistent storage.



### Requirements



- HTML form for task creation.

&nbsp;   

- JavaScript for DOM updates and task state handling.

&nbsp;   

- PHP backend to store tasks in MySQL.

&nbsp;   

- Clear separation between frontend logic and backend processing.

&nbsp;   



### Edge Cases



- Empty task submission

&nbsp;   

- Page reload consistency

&nbsp;   

- Task status synchronization

&nbsp;   



### Deliverables



- `tasks.html`, `tasks.js`, `tasks.php`, `db.sql`
